## Youtube 클론 페이지 만들기

Project for team Query Union

> 쿼리 유니온 (Query Union): 팀장님 성함 '조권호'의 '권'과 팀원 '권정연'님의 '권', 그리고 '유동혁', '이문영' 님의 성에서 느껴지는 연결고리를 활용하여 데이터베이스 쿼리처럼 합쳐진다는 의미를 담았습니다. 프론트엔드에서 데이터를 다루는 작업을 연상시키기도 합니다.
